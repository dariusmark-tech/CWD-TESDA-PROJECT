## 📌 About the Project

This project is the **capstone output** for the Community Web Design (CWD) program offered by TESDA. It demonstrates the core competencies gained throughout the course, including building structured, styled, and interactive web pages from scratch using fundamental web technologies — **HTML, CSS, and JavaScript** — without relying on any external frameworks.

The website is designed to be **fully responsive**, adapting seamlessly across desktops, tablets, and mobile devices.

---

## ✨ Features

- ✅ Fully responsive layout (mobile, tablet, desktop)
- ✅ Smooth navigation and scroll behavior
- ✅ Interactive elements powered by vanilla JavaScript
- ✅ Clean and structured HTML5 semantic markup
- ✅ Custom CSS styling with consistent design system
- ✅ Functional contact form with basic validation
- ✅ Organized gallery/portfolio section
- ✅ Cross-browser compatible

---

## 📄 Pages & Sections

### 🏠 Home / Landing Page
The entry point of the website. Features a hero section with a call-to-action, introducing the purpose of the site and guiding visitors to explore further.

### 👤 About / Services
Provides an overview of the subject — who they are, what they offer, and the services or competencies being highlighted. Structured using clear headings and descriptive content.

### 📬 Contact Form
A user-friendly contact section that allows visitors to get in touch. Includes:
- Input fields for Name, Email, and Message
- Basic form validation using JavaScript
- Submission feedback for the user

### 🖼️ Gallery / Portfolio
A visually organized section showcasing work samples, projects, or images. Designed with a grid layout that remains clean and responsive across all screen sizes.

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| **HTML5** | Page structure and semantic markup |

---

## 📱 Responsive Design

The website uses **CSS Flexbox and Grid** along with **media queries** to ensure a consistent and user-friendly experience across all devices:

| Breakpoint | Target Device |
|---|---|
| `< 480px` | Mobile phones |
| `481px – 768px` | Tablets |
| `769px – 1024px` | Small laptops |
| `> 1024px` | Desktops and large screens |

## 🚀 Getting Started

To view or run the project locally:

### Prerequisites
No installations required. Just a modern web browser (Chrome, Firefox, Edge, Safari).

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/cwd-tesda.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd cwd-tesda
   ```

3. **Open the project**
   - Simply open `index.html` in your browser, or
   - Use **Live Server** in VS Code for a better development experience
